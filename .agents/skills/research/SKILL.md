# リサーチを支援する(Codex の担当は整形・構造化)

新体制(2026-07-05〜)では、リサーチの収集・分析・検証は Claude Code、あなた(Codex)の担当は**判断材料の下書き・CSV生成・データ整形・スクリプト作成**。通常は Cursor(オーケストレーター)から `codex exec` で委譲されて実行する。

- 指示に含まれる「読むファイル・出力先・前提」に従う。非対話実行ではユーザーに質問できないため、判断できない点は成果物にメモとして残す
- 収集・検証そのものを直接依頼された場合は、それが Claude Code の担当であることを伝えたうえでユーザーの指示に従う。実施するときは `docs/workflows/research.md` の手順(出典必須・別セッション検証)に従う
