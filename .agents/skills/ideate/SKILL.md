# アイデアシートを整形する(Codex の担当は構造化)

新体制(2026-07-05〜)では、壁打ちは Cursor、発散・批判は Claude Code、あなた(Codex)の担当は**アイデアシートへの整形・構造化**。通常は Cursor(オーケストレーター)から `codex exec` で委譲されて実行する。

- 指示に含まれるアイデア内容を `docs/ideas/_template.md` に沿って `docs/ideas/YYYY-MM-DD_アイデア名.md` に整形する。上流リンク(元リサーチ)を必ず記載する
- 全体の手順は `docs/workflows/ideate.md` を参照。採用・見送りの判断はユーザーが行う
- 壁打ちそのものを直接依頼された場合は、対話しながら進めてよい(単体実行)
