# ARターゲット / レーザー彫刻検証メモ

## 使うファイル

- `gomi-target.svg`: 印刷・彫刻検証用の高コントラスト画像ターゲット
- `gomi-target.mind`: MindAR compilerで `gomi-target.svg` から生成して、このフォルダへ配置するファイル。生成後にAR画面の追跡が有効になる

## `.mind` 生成手順

1. MindAR Image Targets Compilerを開く
2. `gomi-target.svg` をPNG化するか、ブラウザで表示して画像として保存する
3. 画像をcompilerへ投入し、特徴点の分布を確認する
4. `Download` で取得した `targets.mind` を `gomi-target.mind` にリネームして `public/ar-targets/` に置く
5. `npm run build` して、`dist/ar-targets/gomi-target.mind` が含まれることを確認する

## 印刷時の注意

- 推奨サイズ: 80mm角以上。スマホ検証では100〜120mm角から始める
- マット紙を優先し、光沢紙の反射は避ける
- 周囲に10mm以上の余白を残す
- 折れ、歪み、汚れが出たサンプルは別サンプルとして記録する

## レーザー彫刻時の注意

- 線幅と余白を潰さない。最初は拡大した100mm角以上で試す
- 金属、木材、樹脂など素材ごとにコントラストを記録する
- 彫刻だけでコントラストが足りない場合は、塗料、印刷ラベル、NFC併設へ切り替える
- QR/NFCを併設し、ARが失敗しても同じ作品URLへ戻れるようにする
