# 決定: 施設ローカルマップ × GPS デモ（2件目）

- 日付: 2026-07-13
- ステータス: 確定（最小仕様・技術選定）
- 上流: [技術ベース確定 D1](./2026-07-06_技術ベース確定.md)

## 決定内容

| 項目 | 決定 |
|---|---|
| 位置づけ | 検証用デモ2本のうち2件目。1件目（カメラ認識×ゴミ）と並走可 |
| 体験 | 屋外施設内で、スマホGPSにより自前施設図上の現在地・ゾーンを表示する |
| 技術 | **自前マップ画像 + Geolocation API**。Mapbox / Google Maps API は使わない |
| 配信 | 1件目と同様 GitHub Pages サブパス（実装時に `pages.yml` へ追加） |
| マップ原図 | `docs/assets/fuji-clean-facility-map.png`（藤クリーン リサイクルセンター施設図） |

## スコープ（最小）

- In: 施設図表示、GPS取得、図上マーカー、簡易ゾーン判定・ラベル、拒否/精度不足時の案内、デモ用フォールバック座標
- Out: Mapbox、屋内測位、ナビ/ルート、多人数同期、認証・課金、本番ツアー運用

## 人間作業（ソフトウェア完了後）

- 現地で基準点の緯度経度を数点記録し、画像座標と対応付ける（キャリブレーション）
- 実機での位置精度確認

## 理由

- D1「独自ローカルマップ + GPS」と一致
- 添付の施設イラストをそのまま使える（Mapboxだと衛星図になり、イラスト重ねは結局同等の作業が増える）
- HTTPS 静的配信で足り、APIキー・課金リスクがない
