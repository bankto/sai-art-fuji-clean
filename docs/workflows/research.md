# ワークフロー: リサーチを実施する(主担当: Claude Code)

指定されたテーマ(無ければユーザーに確認)についてリサーチを行い、リサーチノートを作成する。
通常は Cursor(オーケストレーター)から `claude -p` で委譲されて実行する。単体実行(ユーザーが直接依頼)でも手順は同じ。

## モードを選ぶ(最初に必ず)

| モード | 使いどころ | 成果物 |
|---|---|---|
| ライト | 軽い下調べ・裏取り(1〜2時間以内) | `docs/research/YYYY-MM-DD_テーマ名.md`(`_templates/light.md` を使用) |
| 本格 | 意思決定に関わる調査(参考事例・ニッチ探索・比較検討) | `docs/research/YYYY-MM-DD_テーマ名/` フォルダ(`_templates/full/00〜05` を使用) |

迷ったらユーザーに確認する。「この結果で何かを決める」ならば本格モード。

## ライトモードの手順

1. 「何を明らかにするか」の問いを2〜4個立ててユーザーに提示する
2. Web検索や既存資料の調査を行う。事実には必ず出典(URL + 参照日)を付ける
3. `_templates/light.md` に沿ってノートを作成する。「要約(3行以内)」と「示唆(So What)」は省略しない
4. `docs/status.md` の Phase 1 を更新する

## 本格モードの手順(brief → 収集 → 分析 → 検証 → 判断)

```
docs/research/YYYY-MM-DD_テーマ名/
  00_brief.md        # 目的・範囲・調べないこと・判断基準(収集の前に必ず作る)
  01_raw_sources.md  # 収集結果。結論は出さない。出典付き
  02_analysis.md     # 構造化・比較・パターン抽出
  03_fact_check.md   # 別AI・別セッションによる批判的検証
  04_decision.md     # 採用/不採用/保留の判断(ユーザーが決める)
  05_next_actions.md # 次にやること
```

1. **ブリーフ**: `00_brief.md` を作りユーザーに確認する。ブリーフなしで収集を始めない
2. **収集**: 結論を出さずに集める。Claude Code なら専門調査員(`reference-hunter` / `niche-spot-scout` / `official-source-researcher`)が使える
3. **分析**: 比較表・パターン抽出(`02_analysis.md`)。情報源は 公式/準公式/メディア/個人/SNS/推測 に分類する
4. **検証**: **収集したAI・セッションとは別のコンテキスト**でファクトチェックする(Claude Code の `fact-checker`、または別AI)。収集AIに自己検証させない
5. **判断**: `04_decision.md` は判断材料まで整理し、採用/不採用/保留の決定はユーザーが行う
6. `docs/status.md` の Phase 1 を更新する

## 全モード共通の注意

- 出典のない情報を事実として書かない。推測は「推測」と明記する
- 価格・営業状況・開催日・仕様・法律は古くなりやすい → 参照日を必ず記録し、疑わしければ検証する
- 海外事例を含む調査は日本語ソースで止めず、英語・現地語まで当たる
- 会話だけで終わらせない。必ずファイルに残す(ツールを跨いで再利用するため)

## オーケストレーション上の位置づけ

全体の指揮と委譲手順は `docs/workflows/orchestrate.md`(Phase 1 の節)が正本。分担は次のとおり。

- **Cursor**: ブリーフの壁打ち・承認取り付け、収集/検証の段取り、成果物の検証、`docs/status.md` 更新
- **Claude Code**: 収集(reference-hunter / niche-spot-scout / official-source-researcher)と分析
- **Claude Code(別の CLI 呼び出し=別セッション)**: fact-checker による批判的検証。収集と同じ呼び出しに混ぜない
- **Codex**: 判断材料の下書き・CSV化・データ整形
- **ユーザー**: ブリーフの承認と `04_decision.md` の最終判断

委譲されて実行している場合(非対話)はユーザーに質問できない。指示の前提に従い、迷った点は成果物にメモとして残す。
